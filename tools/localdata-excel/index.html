LOCALDATA 종료 후 의원·병원·약국 개·폐업 엑셀 자료실

예전 지방행정인허가데이터개방시스템(LOCALDATA)에서 내려받던 의원·병원·약국 인허가 및 개·폐업 자료를 공공데이터포털 API 기준으로 다시 정리했습니다.

전국 17개 시도별, 업종별, 개업/폐업별 엑셀 파일로 나누어 두었으니 개원·개국 입지 검토에 참고하시면 됩니다. 개원·개국 후보지를 살펴보며 제가 직접 쓰려고 만든 자료를 그대로 공유합니다.

왜 이 자료를 올렸나

그동안 많은 원장님과 약사님이 개원·개국 자리를 알아볼 때 LOCALDATA(지방행정인허가데이터개방시스템)에서 의원·병원·약국 인허가와 개·폐업 현황을 엑셀로 내려받아 확인하셨습니다.

하지만 해당 시스템은 2026년 4월 16일 0시부로 폐쇄되었습니다.

지방행정인허가데이터개방시스템을 이용해주신 모든 분들께 진심으로 감사드립니다. 2026년 4월 16일 0시 부로 지방행정인허가데이터개방시스템은 폐쇄되었습니다. 아울러, '지방행정 인허가정보' 및 '생활편의정보'는 공공데이터포털(data.go.kr)에서 이용하실 수 있습니다.

이제 관련 데이터는 공공데이터포털 Open API를 통해 확인할 수 있습니다. 다만 API 호출 방식이 익숙하지 않은 분들에게는 필요한 자료를 직접 내려받고 정리하는 과정이 번거로울 수 있습니다.

예전처럼 엑셀 파일로 한눈에 확인하는 방식이 더 편한 분들이 많을 것 같아, 공공데이터포털 API에서 받은 자료를 시도·업종·상태별 엑셀로 정리해 올려둡니다.

이용 전 참고사항

이 자료는 공공데이터포털에서 제공하는 지방행정 인허가정보를 바탕으로 정리한 참고용 자료입니다.

인허가일자 기준으로 정리했으며, 지자체 신고 이후 공공데이터에 반영되기까지 시차가 있을 수 있습니다. 특히 최근 1~2개월 자료는 일부 누락되거나 반영이 지연될 수 있습니다.

시도별 다운로드

시도를 펼치면 의원·병원·약국의 개업/폐업 엑셀 파일을 받을 수 있습니다. 괄호 안 숫자는 해당 시도 파일 전체에 포함된 의원·병원·약국 데이터 건수입니다.
